Acerca de mi

Soy un desarrollador web apasionado por el aprendizaje constante y la implementación de nuevas tecnologías para crear proyectos de alta complejidad. Con experiencia en JavaScript, CSS, HTML y React, he enfocado mi carrera en desarrollar soluciones eficientes que mejoren la calidad de los productos y optimicen los tiempos de desarrollo. Desde que comencé en el mundo de la programación, he trabajado para entender y resolver los problemas de manera efectiva, buscando siempre métodos y tecnologías modernas que garanticen el éxito de cada proyecto. Mi enfoque es que la programación no solo se trata de escribir líneas de código, sino de comprender las necesidades reales de una organización y cómo la tecnología puede ser utilizada para ofrecer soluciones efectivas y aportar valor tangible.

Creo firmemente que el proceso de desarrollo debe ir más allá de la solución técnica, integrándose con los objetivos estratégicos de la empresa. Mi objetivo es ser un motor de cambio, utilizando mis habilidades para mejorar los procesos y facilitar el crecimiento de la organización a través de la innovación y la eficiencia.

Proyectos

Aquí algunos de los proyectos en los que he trabajado, donde aplico mis conocimientos en tecnologías modernas para crear soluciones prácticas y escalables.

Pokedex

App de React que usa useContext, useReducer, Global State y React Router Dom. Consume la API de Pokedex para mostrar información de Pokémon, gestionando estados globales y navegación entre vistas.

Users CRUD

App de React que utiliza useState y useEffect para manejar estados y efectos. Realiza peticiones HTTP (GET, POST, PUT, PATCH, DELETE) a una API, y usa react-hook-form para gestionar formularios. Además, aprovecha la propiedad children para componer componentes reutilizables.

Rick and Morty App

App de React que utiliza useState y useEffect para gestionar estados y efectos. Muestra listas de personajes de la API de Rick and Morty con renderizado condicional y usa useRef para manejar referencias. Consume la API mediante un custom hook con fetch y gestiona errores en las peticiones.

Weather App

App de React que usa useState y useEffect para manejar el estado y efectos. Realiza peticiones a la API de Open Weather Map para obtener información sobre el clima y la muestra dinámicamente en la interfaz. Utiliza fetch para consumir la API y actualizar los datos en tiempo real.
