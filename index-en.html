About me

I am a web developer passionate about constant learning and implementing new technologies to create highly complex projects. With experience in JavaScript, CSS, HTML, and React, I have focused my career on developing efficient solutions that improve product quality and optimize development times. Since I began in the world of programming, I have worked to understand and solve problems effectively, always seeking modern methods and technologies that guarantee the success of each project. My approach is that programming is not just about writing lines of code, but about understanding the real needs of an organization and how technology can be used to offer effective solutions and provide tangible value.

I firmly believe that the development process must go beyond the technical solution, integrating it with the company's strategic objectives. My goal is to be a driver of change, using my skills to improve processes and facilitate organizational growth through innovation and efficiency.

Projects

Here are some of the projects I've worked on, where I apply my knowledge of modern technologies to create practical and scalable solutions.

Pokedex

React app that uses useContext, useReducer, Global State, and React Router Dom. It consumes the Pokedex API to display Pokémon information, manage global state, and navigate between views.

Users CRUD

React app that uses useState and useEffect to manage states and effects. It makes HTTP requests (GET, POST, PUT, PATCH, DELETE) to an API and uses react-hook-form to manage forms. It also leverages the children property to compose reusable components.

Rick and Morty App

React app that uses useState and useEffect to manage states and effects. It displays lists of characters from the Rick and Morty API with conditional rendering and uses useRef to handle references. It consumes the API using a custom hook with fetch and handles errors on requests.

Weather App

React app that uses useState and useEffect to manage state and effects. It makes requests to the Open Weather Map API to obtain weather information and displays it dynamically on the interface. It uses fetch to consume the API and update data in real time.
